# DB 계층 노드 상세 템플릿

> db(테이블/엔티티) 계층 step 의 상세 섹션. 사람이 편집하는 권위 — 런타임 로드(재빌드 불필요).
> 형식: `## <라벨> {#<id>}` = 섹션, 그 아래 본문 = LLM 채움 지시(promptHint, 산문).
> 각 섹션은 step slice(스키마 정의/@Entity/DDL)에서 인용해 채운다(근거 ≥1).

## 역할 {#role}
이 데이터 노드가 흐름에서 맡는 역할을 한 문단으로 — 어떤 데이터를 보관/표현하는지.

## 스키마 {#schema}
테이블/엔티티의 주요 컬럼·키·제약·관계. 스키마 정의/@Entity/DDL 에서 인용.
