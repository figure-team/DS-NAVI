# Other(기타) 계층 노드 상세 템플릿

> 계층 미분류(unknown) step — 도메인 엔터티/데이터 구조/협력 객체. 사람이 편집하는 권위 — 런타임 로드(재빌드 불필요).
> 형식: `## <라벨> {#<id>}` = 섹션, 그 아래 본문 = LLM 채움 지시(promptHint, 산문).
> 각 섹션은 step slice(클래스 필드 선언/검증 로직)에서 인용해 채운다(근거 ≥1).
> 파일명 other.md = 코드 계층 키 unknown 에 매핑된다.

## 역할 {#role}
이 노드가 흐름에서 맡는 역할을 한 문단으로 — 데이터 구조 또는 협력 객체로서.

## 데이터 구조 {#dataShape}
주요 필드/구조와 지켜야 할 불변식. 클래스 필드 선언/검증 로직에서 인용.
