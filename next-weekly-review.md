## 1. 금주 성과 — 커밋에 안 드러난 것
- 분석 데이터 생산 라인을 하나로 통일하기로 결정하고 완료 — 그동안 두 갈래(구형 AI 전수 분석 / 신형 업무 지도 분석)로 만들어져 화면마다 기준 시점이 달랐던 데이터를 신형 하나로 정리, 모든 화면이 같은 시점·같은 근거의 데이터를 보게 됨
- 대형 실프로젝트 검증 완료 — 공공 표준프레임워크(6,100여 파일)와 엠모바일 실서비스 코드를 끝까지 분석 완주(중간에 사용량 한도로 끊겨도 이어서 재개되는 체계 확립). 엠모바일 업무 도메인은 13개 그룹·84개 업무로 계층 정리해 검수 가능한 상태
- '구조' 화면을 업무 중심 4단계 탐색(시스템 전체 → 업무 그룹 → 세부 업무 → 업무 흐름도)으로 재구축 — 그래프 디자인은 여러 시안을 화면 비교한 끝에 오픈소스 원형(understand-anything 데모)의 도메인 그래프 스타일로 확정했고, 별도 메뉴였던 '구조'는 '업무 지도' 안 탭으로 통합(메뉴 수 축소 결정)
- 그래프 판독성 집중 개선 — 카드·선이 겹쳐 관계를 읽을 수 없던 문제를 해결(실제 크기 기반 배치, 직각 선으로 카드 회피, 양방향 관계는 한 줄로 병합, 카드에 마우스를 올리면 그 카드의 연결만 강조). 선을 클릭하면 "어떤 파일 때문에 연결됐는지" 근거가 오른쪽 패널로 열림
