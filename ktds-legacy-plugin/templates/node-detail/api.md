# API 계층 노드 상세 템플릿

> api(엔드포인트/컨트롤러) 계층 step 의 상세 섹션. 사람이 편집하는 권위 — 런타임 로드(재빌드 불필요).
> 형식: `## <라벨> {#<id>}` = 섹션, 그 아래 본문 = LLM 채움 지시(promptHint, 산문).
> 각 섹션은 step slice(핸들러 메서드/검증 코드)에서 인용해 채운다(근거 ≥1).
> 메서드·호출관계는 결정론(엔진 보유)이라 섹션이 아니다.

## 역할 {#role}
이 흐름에서 이 엔드포인트/컨트롤러가 맡는 역할을 한 문단으로 — 무엇을 받아 어디로 위임하는지.

## 요청 처리 {#request}
받는 요청 파라미터·검증·인증/권한 처리. 핸들러 메서드 시그니처와 검증 코드에서 인용.
